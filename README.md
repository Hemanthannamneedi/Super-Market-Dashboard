# 🛒 Supermarket Sales Dashboard

## 📌 Overview
This repository contains a Power BI dashboard designed to analyze and visualize retail sales performance. The `SUPER MARKET SALES DASHBOARD.pbix` file provides comprehensive data insights to track revenue streams, understand customer purchasing trends, and optimize supermarket operations.

## 📊 Key Features
* **Sales Tracking:** High-level key performance indicators (KPIs) monitoring total sales, gross income, and profit margins.
* **Product Line Analysis:** Visual breakdowns of top-performing and underperforming product categories.
* **Customer Demographics:** Insights into customer types (e.g., Member vs. Normal), gender distributions, and their impact on revenue.
* **Time-Series Trends:** Sales performance analyzed by date, time of day, and seasonal peaks.
* **Interactive Filtering:** Dynamic slicers allowing users to drill down by branch location, city, and payment method.

## 🛠️ Tools & Technologies
* **Power BI Desktop:** Used for data modeling, DAX measure creation, and visualization.
* [cite_start]**Theme:** Utilizes the built-in "Frontier" theme for a professional and consistent visual layout[cite: 58].

## 🚀 Getting Started
1. **Install Power BI:** Download Power BI Desktop from the [Microsoft Store](https://powerbi.microsoft.com/desktop/) if you don't already have it.
2. **Download the File:** Clone this repository or download the `SUPER MARKET SALES DASHBOARD.pbix` file directly.
3. **Open the Dashboard:** Double-click the file to launch it in Power BI Desktop.
4. **Interact:** Click on the charts and use the slicers to cross-filter and explore specific data segments.
